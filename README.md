# 원티드 프리온보딩 프론트엔드 - 선발 과제


#### ✏️ 성명 : 김희민 

#### ✏️  프로젝트의 실행 방법
1. 회원 가입 (유효성 검사 통과 시에만 회원가입 버튼 클릭 가능)
2. 로그인 페이지에서 로그인
3. 투두 페이지에서 할 일  __*추가/수정/삭제/완료*__ 표시 가능
   
   - 수정 버튼 클릭 시 수정 모드(버튼이 수정/삭제 -> 제출/취소로 변경)
   - 수정 작업 후 제출 버튼 클릭 시 할 일 수정 완료
   - 완료 체크 시 수정 모드 비활성화(수정/삭제 버튼 비활성화)
